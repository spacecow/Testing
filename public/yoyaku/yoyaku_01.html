Before doing the migration, if there exists a root password for the
MySQL database, that password must entered in the configuration file:


yoyaku/config/database.yml

  ...

development:
  ...
  username: root
  password: root
  ...

test:
  ...
  username: root
  password: root
  ...

production:
  ...
  username: root
  password: root
  ...


Further, the databases must be created:


yoyaku>rake db:create:all
(in D:/ruby/rails/yoyaku)
