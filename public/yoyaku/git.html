You can see the version number with:

yoyaku>git --version
git version 1.6.3.2.1299.gee46c



yoyaku>git config --global user.name "Johan Sveholm"
fatal: $HOME not set


The HOME variable is not set as you can see:

>set home
HOMEDRIVE=C:
HOMEPATH=\Documents and Settings\Johan


That can be set by right-clicking on My Computer and choose
Properties->Advanced->Environment Variables and add HOME if it is not
added already. Set it to %HOMEDRIVE%%HOMEPATH% in order to point it
somewhere below C:\Documents and Settings. This is where Git stores
the .gitconfig file. 

Now check home again:

>set home
HOME=C:\Documents and Settings\Johan
HOMEDRIVE=C:
HOMEPATH=\Documents and Settings\Johan


And set:

yoyaku>git config --global user.name "Johan Sveholm"
yoyaku>git config --global user.email 'jsveholm@gmail.com'
yoyaku>git config --global alias.co checkout
