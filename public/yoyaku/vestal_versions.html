Vestal Versions


In order to make vestal version operate on single attributes,
(in this case note) add this piece of code to the original file.
It is not pretty, but until the author himself has fixed the problem,
it works:


c:/ruby/lib/ruby/gems/1.8/gems/laserlemon-vestal_versions-0.6.0/lib/vestal_versions.rb

require 'version'

module LaserLemon
  module VestalVersions
		...

    module ClassMethods
      ...
    end

    module InstanceMethods
      private
        ...

        def create_version
          versions.create(:changes => changes, :number => (last_version + 1)) \
            if changes["note"]
          reset_version
        end

      public
        ...
    end
  end
end

ActiveRecord::Base.send(:include, LaserLemon::VestalVersions)
